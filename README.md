# This project is the result of React Native test

## Using:
- Typescript
- Hook
- React Native WebView

# How to run?

Android:
```sh
yarn install
npx react-native start --reset-cache
npx react-native run-android
```

IOS:
```sh
yarn install
cd ios && pod install
cd ..
npx react-native start --reset-cache
npx react-native run-ios
```
